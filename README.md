# Thefacebook Clone 2004

Este proyecto es un clon funcional de Thefacebook 2004, desarrollado en PHP y MySQL.  
Permite registro, login y visualización de información de usuarios, manteniendo el diseño original.

## Tecnologías usadas
- PHP 8
- MySQL (InfinityFree)
- HTML5 / CSS3

## Instalación
1. Subir todos los archivos a un hosting que soporte PHP y MySQL.
2. Configurar `config/database.php` con tus datos de la base de datos.
3. Abrir `index.php` en el navegador.

## Uso
- Registro de usuarios: `register.php`
- Login de usuarios: `index.php`
- Home: `home.php`
